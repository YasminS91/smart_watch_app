Description of App

With such a small screen, reading on a smartwatch the conventional way could be difficult. By printing one word to the screen at a time in quick succession, our application would implement Rapid Serial Visual Representation, an alternative method of reading that would be ideal for the small screen of a smartwatch. RSVR implementations such as Spritz and Spreeder have been created before, but, to our group’s knowledge, one hasn’t been created for the smartwatch. Our application would access text files from the smartwatch’s file system and present them to the user with easily customisable speeds and fonts. What makes our application good for our group is that it is easily scalable, significantly decreasing our risk of failure. It could start out by interpreting .txt files before moving on to pdf, docx, and epub files along with other ebook formats. Ideally, we hope to eventually give our app HTML-reading capabilities, opening up the possibility of creating a primitive browser for the smartwatch.

Why GitHub?

GitHub offers all of the distributed revision control (allows many software developers to work on a project without requiring them to share a common network) and source code management (SCM) functionality of Git as well as adding its own features.
Source control or revision control is the management of changes to documents, web sites, computer programs and information. It provides the ability of reverting a document to a previous revision, which is critical for allowing editors to track each other's edits, correct mistakes, and defend against vandalism and spamming.
GitHub provides a Web-based graphical interface and desktop as well as mobile integration.
It also provides access control (selective authorization to access resources) and several collaboration features such as bug tracking (keeps track of reported software bugs/updated about new discovered bugs), feature requests, task management, wikis for every project. Add to that documentation, issue tracking, search engine, nested tasks-list (time management) within files, hosting small websites from public repositories on GitHub, specified graphs, directory of integration, commits history email notifications, etc.
By using GitHub, time and space required to install software on probably thousands of client computers.
GitHub is a client-server software application that the client runs in a web browser.
GitHub allows registered and non-registered users to browse public repositories on the site. Registered users can discuss, manages, create repositories, submit their contribution to others’ repositories and review changes to code.
GitHub provides functions that perform as a social network, such as followers, feed, wiki, a a graph that keeps track of developers’ work and shows what version of work is the newest.
GitHub offers both plans for private repositories and free accounts, which are usually used to host open-source software projects.
